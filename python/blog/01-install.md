## 源码安装

```shell
0. yum -y install zlib*
1. wget https://www.python.org/ftp/python/3.8.0/Python-3.8.0.tgz; tar -zxvf .tgzta
2. ./configure --prefix=/usr/local/python3
3. make;make install
4. ln -sf /usr/local/python3/bin/python3 /usr/bin/python3
   ln -sf /usr/local/python3/bin/pip3 /usr/bin/pip3
```
